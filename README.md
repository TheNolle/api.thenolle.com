# Nolly's API 🚀

A public, community-driven REST API built with Express, MongoDB, and ReactJS. Nolly's API offers free access to a wide range of functionalities, with data provided and updated by users.



## 🌟 Summary
Nolly's API aims to create a collaborative platform where users contribute to and maintain a multi-functional API. It is built using Express and MongoDB on the backend, and ReactJS on the frontend. Users can access various API endpoints through HTTP requests.

## 📦 Installation
1. Clone the repository:
```bash
git clone https://github.com/thenolle/api.thenolle.com.git
```

2. Install dependencies:
```bash
cd api.thenolle.com
npm install
```


3. Set up configuration:
```bash
cp config.json.template config.json
```
Fill in the required variables in the `config.json` file.

4. Start the server:
```bash
npm start
```
The API will now be running on the specified port.


## 🔧 Usage

To access the various API endpoints, make HTTP requests to:
<http://localhost:{port}/{endpoint}>

Replace `{endpoint}` with the desired API endpoint.

Please refer to the [API documentation](https://github.com/thenolle/api.thenolle.com/wiki) for a list of available endpoints and detailed usage instructions.

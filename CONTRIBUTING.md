# Contributing to Nolly's API

Thank you for considering contributing to Nolly's API! We appreciate your help in improving and expanding the project.


## Getting Started
1. Fork the repository and clone it to your local machine.
2. Install dependencies by running `npm install`.
3. Make your changes and test them.
4. Commit your changes and create a pull request.


## Reporting Bugs
If you encounter any bugs, please [create a new issue](https://github.com/thenolle/nollys-api/issues/new) with a detailed description of the problem and steps to reproduce it.


## Requesting Features
If you have ideas for new features, please [create a new issue](https://github.com/thenolle/nollys-api/issues/new) with a detailed description of your proposal.


## Pull Requests
When submitting a pull request, please ensure your code adheres to our coding guidelines and is well-tested.  
Please provide a clear and concise description of your changes in the pull request description.


## Code of Conduct
By participating in this project, you agree to abide by the terms of our [Code of Conduct](CODE_OF_CONDUCT.md).
